# 🚀 Deploy CogOS Site to Vercel

Your CogOS landing page is ready! Here are 3 ways to deploy it to Vercel (all free):

## Option 1: Drag & Drop (Easiest)

1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag the `cogos-site` folder onto the page
3. Click "Deploy"
4. Done! Your site will be live at `cogos-site.vercel.app`

## Option 2: Vercel CLI (Fastest)

```bash
# From the cogos-site directory:
cd cogos-site

# Login to Vercel (opens browser)
vercel login

# Deploy
vercel --yes

# Your site will be live at: https://cogos-site-xxx.vercel.app
```

## Option 3: Connect GitHub (Best for Updates)

1. Push `cogos-site` to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import your GitHub repo
4. Vercel auto-deploys on every push
