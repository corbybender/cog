Getting Started

Up and running
in three steps.

No account. No cloud. No vendor lock-in. Everything executes locally on your machine.

01

Install CogOS

Install via pip. All 40+ modules and 10 cognitive agents are included by default. pip install cogos

02

Integrate into your project

Run the installer from your project root. CogOS scans your stack and configures itself automatically. python install_cogos.py

03

Launch the Web UI (optional)

A local dashboard for task management, analytics, and module browsing. cd cogos/web-ui && ./start.sh
