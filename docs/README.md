## 🎯 Common Tasks

### Install CogOS
```bash
pip install cogos
python -m cogos.install_cogos
```

### Use Multi-Agent Intelligence
```python
from cogos import CogOS
cogos = CogOS()
result = cogos.think("Build a REST API")
```

### Create Custom Module
```bash
cogos create-module my-module
```

### Run Tests
```bash
pytest
```
